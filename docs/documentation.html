Documentation

The document hub for Freally Central — what it is, how it works, and the project's roadmap and build guide.

What is Freally Central?

Freally Central is the brand's hub: one desktop app (and an embeddable panel that ships inside every other Freally app) that shows the whole product family in a card grid — big icons, per-OS downloads, live install/update status, changelogs & "What's New", real download counts, and a Download All button — with a live 0–100 % download/install progress bar and hands-off (silent) installation, all in 18 languages.

  • Data-driven grid — the product list comes from a hosted catalog manifest, so it updates without shipping a new Central build.
  • Real numbers only — download counts come from GitHub's release API and the progress percent tracks real bytes; nothing is fabricated.
  • Consent & safety — installs run only on your click, only on official verified installers, and request elevation at most once per batch.
  • Cross-platform — built and tested on Windows, macOS, and Linux from Phase 0's CI matrix.

How it works

  • Browse — every Freally app as a card: icon, tagline, features, start date, latest version + date.
  • Status — each card shows Installed ✓ / Update available / Not installed.
  • Download — pick your OS's installer, or Download All; watch the precise 0–100 % bar.
  • Install — installers run unattended, accepting their defaults, then the card flips to Installed with an Open button.
  • Changelog & What's New — read each app's latest release notes in-app.

Project documents
