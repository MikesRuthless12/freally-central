What it is

Freally Central connects you to every Freally-brand app from one place. Browse the family, see what's installed and what needs an update, read each app's changelog and "What's New", and download or install with a live progress bar — including a one-click Download All. It also ships as an embeddable panel inside the other Freally apps, so the whole family is reachable from anywhere. Local-first and honest: every download number is a real GitHub statistic, and installs run only on your click.
